# AIML-Task-2-Exploratory-Data-Analysis
Understand data using statistics and visualizations.

This project focuses on Exploratory Data Analysis (EDA) of the Titanic dataset using Python in Google Colab. The primary objective is to understand the structure, quality, and characteristics of the data before applying any machine learning models. EDA helps uncover hidden patterns, relationships, trends, and anomalies within the dataset, enabling better decision-making during the data preprocessing and modeling stages.

The analysis was performed using Pandas for data loading, cleaning, and statistical analysis, NumPy for numerical computations, Matplotlib and Seaborn for creating visualizations such as histograms, boxplots, count plots, pairplots, and correlation heatmaps, and Plotly for interactive visual exploration. The workflow included dataset inspection, missing value analysis, generation of summary statistics (mean, median, standard deviation, variance, skewness, and kurtosis), distribution analysis of numerical features, outlier detection, correlation analysis, and feature relationship exploration.

The analysis revealed several important insights about the Titanic dataset. Female passengers had significantly higher survival rates than male passengers, and passengers traveling in First Class were more likely to survive than those in lower classes. The Fare feature contained several extreme values, indicating the presence of outliers, while the Age feature showed a concentration of passengers between 20 and 40 years old. Missing values were identified in columns such as Age, Cabin, and Embarked, highlighting the need for preprocessing before model training. Correlation analysis showed meaningful relationships between passenger class, fare, and survival status. Overall, the project successfully demonstrated how EDA can be used to understand data quality, discover patterns, identify anomalies, and generate actionable insights for future machine learning tasks.
