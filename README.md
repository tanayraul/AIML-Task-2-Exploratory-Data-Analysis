# AIML-Task-2-Exploratory-Data-Analysis
Understand data using statistics and visualizations.
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python in Google Colab. The goal is to understand the dataset through statistical analysis and data visualization before applying machine learning techniques.

The analysis includes dataset inspection, missing value detection, summary statistics generation, histogram and boxplot visualizations, correlation analysis, pairplots, and survival-based insights. Libraries used include Pandas, NumPy, Matplotlib, Seaborn, and Plotly.

Key findings show that survival rates were strongly influenced by passenger gender and class, fare values contained significant outliers, and several features contained missing values requiring attention during preprocessing. The project demonstrates essential data analysis skills such as feature exploration, pattern identification, trend analysis, and insight generation.
